This page is the public entry point for the 2026 Cyberman HW-CTF materials. From here, you can access the event documentation and the browser-based firmware flasher for the Cyberman boards.

Available resources

The documentation section includes guides, setup notes, and supporting material for participants. The flasher provides a browser-based way to install compatible firmware images onto the event hardware.
