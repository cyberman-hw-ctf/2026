Firmware Extraction using esp32knife_fixed¶

Prerequisites¶

  • Clone the repository: esp32knife_fixed
  • ESP32-C3 device connected via USB
  • Python 3.x environment

Steps¶

  1. Install dependencies from the repository's requirements
  2. Connect your ESP32-C3 to the system
  3. Run the extraction tool following the repo's instructions to dump the firmware
  4. Extract and save the firmware binary to your working directory

Analyzing Firmware with Ghidra¶

Initial Setup¶

  • Open Ghidra and create a new project
  • Import the extracted firmware binary
  • Select correct processor architecture ?

Analysis Steps¶

  1. Disassemble the firmware and let Ghidra perform initial analysis
  2. Identify key functions using the Symbol Tree and Listing windows
  3. Search for strings that hint at CTF flags or sensitive data
  4. Use cross-references to trace flag-related function calls
  5. Examine memory sections for hardcoded flag data

Flag Extraction Tips¶

  • Look for string constants containing format patterns (CYBERMAN{...})
  • Check .rodata sections for embedded text
  • Trace cryptographic or encoding/decoding routines
