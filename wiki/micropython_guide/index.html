MicroPython on the ESP32-C3¶

This board can be flashed with MicroPython, which lets you write pythonic code that runs directly on the custom ESP32-C3 board. This guide shows how to:

  1. Flash MicroPython onto the board
  2. Connect to it from your computer
  3. Upload Python scripts
  4. Run code automatically on boot

Quick start¶

pip install --user mpremote
mpremote connect auto fs ls
mpremote connect auto fs cp main.py :main.py
mpremote connect auto reset

What you need - The ESP32-C3 board - USB cable with data support - Computer with Python installed - Serial-capable USB port

1. Flash MicroPython¶

If the board is already flashed with MicroPython, you can skip this    step. Flash the MicroPython firmware from the web serial or from the official website.

2. Install the required tool¶

Download mpremote, which is the standard MicroPython command-line tool.

pip install mpremote

3. Find the serial port¶

Typical serial device names: - Linux:/dev/ttyACM0 or /dev/ttyUSB0 - macOS:/dev/cu.usbmodem... - Windows:COM3, COM4, etc. Test connection: mpremote connect auto fs ls If needed, specify manually: mpremote connect /dev/ttyACM0 fs ls

4. Open the MicroPython REPL¶

mpremote connect auto repl

Example: print("Hello from ESP32-C3")

Exit REPL with: Ctrl+]

5. Upload a script¶

Example blink.py: mpremote connect auto fs cp blink.py :blink.py   The : means the device filesystem.

6. Run a script without saving¶

mpremote connect auto run blink.py  

7. Run code automatically on boot¶

MicroPython uses: - boot.py - main.py Upload: 

mpremote connect auto fs cp main.py :main.py
mpremote connect auto reset

8. Upload a library folder¶

mpremote connect auto fs mkdir :lib
mpremote connect auto fs cp -r lib/. :lib/
 

9. Useful commands¶

List files: mpremote connect auto fs ls

View file: mpremote connect auto fs cat :main.py   Delete file: mpremote connect auto fs rm :main.py   Reset board: mpremote connect auto reset  

10. Minimal example using the 5 on-board LEDs¶

main.py:

from machine import Pin
from neopixel import NeoPixel
from time import sleep

NUM_LEDS = 5
PIN_NUM = 5

np = NeoPixel(Pin(PIN_NUM), NUM_LEDS)

while True:
for i in range(NUM_LEDS): # Red
np[i] = (20, 0, 0)
np.write()
sleep(0.5)
for i in range(NUM_LEDS): # Green
np[i] = (0, 20, 0)
np.write()
sleep(0.5)
for i in range(NUM_LEDS): # Blue
np[i] = (0, 0, 20)
np.write()
sleep(0.5)
for i in range(NUM_LEDS): # Off
np[i] = (0, 0, 0)
np.write()
sleep(0.5)

Upload: mpremote connect auto fs cp main.py :main.py  mpremote connect auto reset  

Troubleshooting¶

Board not detected¶

  • try another USB cable
  • reconnect the board
  • check the serial port name

Permission denied on Linux¶

sudo usermod -a -G dialout $USER Log out and log back in.

Code crashes on boot¶

Your main.py may contain an error. Replace or remove it using mpremote.
